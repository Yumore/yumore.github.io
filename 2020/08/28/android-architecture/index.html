    MVC


        ViewпјҡXMLеёғеұҖж–Үд»¶
    Modelпјҡе®һдҪ“жЁЎеһӢпјҲж•°жҚ®зҡ„иҺ·еҸ–гҖҒеӯҳеӮЁгҖҒж•°жҚ®зҠ¶жҖҒеҸҳеҢ–пјү
    ControllerпјҡеҜ№еә”дәҺActivityпјҢеӨ„зҗҶж•°жҚ®гҖҒдёҡеҠЎе’ҢUI

        дјҳзӮ№пјҡз»“жһ„жё…жҷ°пјҢдҪҺиҖҰеҗҲпјҢжңүеҲ©дәҺз»„д»¶йҮҚз”Ёпјӣ
    зјәзӮ№пјҡActivity/Fragmentдёӯд»Јз ҒиҫғеӨҡпјҢз»“жһ„жҜ”иҫғеӨҚжқӮпјӣ
    1. з”ЁжҲ·еҸҜд»Ҙеҗ‘ View еҸ‘йҖҒжҢҮд»ӨпјҲDOM дәӢд»¶пјүпјҢеҶҚз”ұ View зӣҙжҺҘиҰҒжұӮ Model ж”№еҸҳзҠ¶жҖҒгҖӮ
    2. з”ЁжҲ·д№ҹеҸҜд»ҘзӣҙжҺҘеҗ‘ Controller еҸ‘йҖҒжҢҮд»ӨпјҲж”№еҸҳ URL и§ҰеҸ‘ hashChange дәӢд»¶пјүпјҢеҶҚз”ұ Controller еҸ‘йҖҒз»ҷ ViewгҖӮ
    3. Controller йқһеёёи–„пјҢеҸӘиө·еҲ°и·Ҝз”ұзҡ„дҪңз”ЁпјҢиҖҢ View йқһеёёеҺҡпјҢдёҡеҠЎйҖ»иҫ‘йғҪйғЁзҪІеңЁ ViewгҖӮжүҖд»ҘпјҢBackbone зҙўжҖ§еҸ–ж¶ҲдәҶ ControllerпјҢеҸӘдҝқз•ҷдёҖдёӘ RouterпјҲи·Ҝз”ұеҷЁпјү гҖӮ

    MVP е°ұиғҪеҫҲеҘҪең°и§ЈеҶіиҝҷдёӘй—®йўҳпјҢе…¶ж ёеҝғзҗҶеҝөжҳҜйҖҡиҝҮдёҖдёӘжҠҪиұЎзҡ„ View жҺҘеҸЈпјҲдёҚжҳҜзңҹжӯЈзҡ„ View еұӮпјүе°Ҷ Presenter дёҺзңҹжӯЈзҡ„ View еұӮиҝӣиЎҢи§ЈиҖҰгҖӮPersenter жҢҒжңүиҜҘ View жҺҘеҸЈпјҢеҜ№иҜҘжҺҘеҸЈиҝӣиЎҢж“ҚдҪңпјҢиҖҢдёҚжҳҜзӣҙжҺҘж“ҚдҪң View еұӮгҖӮиҝҷж ·е°ұеҸҜд»ҘжҠҠи§Ҷеӣҫж“ҚдҪңе’ҢдёҡеҠЎйҖ»иҫ‘и§ЈиҖҰпјҢд»ҺиҖҢи®© Activity жҲҗдёәзңҹжӯЈзҡ„ View еұӮгҖӮ

    дҪҶ MVP д№ҹеӯҳеңЁдёҖдәӣејҠз«Ҝпјҡ

    1. PresenterпјҲд»ҘдёӢз®Җз§° PпјүеұӮдёҺ ViewпјҲд»ҘдёӢз®Җз§° VпјүеұӮжҳҜйҖҡиҝҮжҺҘеҸЈиҝӣиЎҢдәӨдә’зҡ„пјҢжҺҘеҸЈзІ’еәҰдёҚеҘҪжҺ§еҲ¶гҖӮзІ’еәҰеӨӘе°ҸпјҢе°ұдјҡеӯҳеңЁеӨ§йҮҸжҺҘеҸЈзҡ„жғ…еҶөпјҢдҪҝд»Јз ҒеӨӘиҝҮзўҺзүҲеҢ–пјӣзІ’еәҰеӨӘеӨ§пјҢи§ЈиҖҰж•ҲжһңдёҚеҘҪгҖӮеҗҢж—¶еҜ№дәҺ UI зҡ„иҫ“е…Ҙе’Ңж•°жҚ®зҡ„еҸҳеҢ–пјҢйңҖиҰҒжүӢеҠЁи°ғз”Ё V еұӮжҲ–иҖ… P еұӮзӣёе…ізҡ„жҺҘеҸЈпјҢзӣёеҜ№жқҘиҜҙзјәд№ҸиҮӘеҠЁжҖ§гҖҒзӣ‘еҗ¬жҖ§гҖӮеҰӮжһңж•°жҚ®зҡ„еҸҳеҢ–иғҪиҮӘеҠЁе“Қеә”еҲ° UIгҖҒUI зҡ„иҫ“е…ҘиғҪиҮӘеҠЁжӣҙж–°еҲ°ж•°жҚ®пјҢйӮЈиҜҘеӨҡеҘҪпјҒ
    2. MVP жҳҜд»Ҙ UI дёәй©ұеҠЁзҡ„жЁЎеһӢпјҢжӣҙж–° UI йғҪйңҖиҰҒдҝқиҜҒиғҪиҺ·еҸ–еҲ°жҺ§д»¶зҡ„еј•з”ЁпјҢеҗҢж—¶жӣҙж–° UI зҡ„ж—¶еҖҷиҰҒиҖғиҷ‘еҪ“еүҚжҳҜеҗҰжҳҜ UI зәҝзЁӢпјҢд№ҹиҰҒиҖғиҷ‘ Activity зҡ„з”ҹе‘Ҫе‘ЁжңҹпјҲжҳҜеҗҰе·Із»Ҹй”ҖжҜҒзӯүпјүгҖӮ
    3. MVP жҳҜд»Ҙ UI е’ҢдәӢд»¶дёәй©ұеҠЁзҡ„дј з»ҹжЁЎеһӢпјҢж•°жҚ®йғҪжҳҜиў«еҠЁең°йҖҡиҝҮ UI жҺ§д»¶еҒҡеұ•зӨәпјҢдҪҶжҳҜз”ұдәҺж•°жҚ®зҡ„ж—¶еҸҳжҖ§пјҢжҲ‘д»¬жӣҙеёҢжңӣж•°жҚ®иғҪиҪ¬иў«еҠЁдёәдё»еҠЁпјҢеёҢжңӣж•°жҚ®иғҪжӣҙжңүжҙ»жҖ§пјҢз”ұж•°жҚ®жқҘй©ұеҠЁ UIгҖӮ
    4. V еұӮдёҺ P еұӮиҝҳжҳҜжңүдёҖе®ҡзҡ„иҖҰеҗҲеәҰгҖӮдёҖж—Ұ V еұӮжҹҗдёӘ UI е…ғзҙ жӣҙж”№пјҢйӮЈд№ҲеҜ№еә”зҡ„жҺҘеҸЈе°ұеҝ…йЎ»еҫ—ж”№пјҢж•°жҚ®еҰӮдҪ•жҳ е°„еҲ° UI дёҠгҖҒдәӢд»¶зӣ‘еҗ¬жҺҘеҸЈиҝҷдәӣйғҪйңҖиҰҒиҪ¬еҸҳпјҢзүөдёҖеҸ‘иҖҢеҠЁе…Ёиә«гҖӮеҰӮжһңиҝҷдёҖеұӮд№ҹиғҪи§ЈиҖҰе°ұжӣҙеҘҪдәҶгҖӮ
    5. еӨҚжқӮзҡ„дёҡеҠЎеҗҢж—¶д№ҹеҸҜиғҪдјҡеҜјиҮҙ P еұӮеӨӘеӨ§пјҢд»Јз ҒиҮғиӮҝзҡ„й—®йўҳдҫқз„¶дёҚиғҪи§ЈеҶігҖӮ

    MVP



        Module:ж•°жҚ®еӨ„зҗҶеұӮ
    View:и§ҶеӣҫеұӮ
    Prsenter:д»ЈзҗҶеұӮпјҢи§ЈеҶіviewеұӮе’ҢcontrollerеұӮзҡ„иҖҰеҗҲ

        дјҳзӮ№пјҡз»“жһ„жё…жҷ°пјҢдҪҺиҖҰеҗҲпјҢд»Јз Ғжҳ“з»ҙжҠӨпјҢйҖӮеҗҲеӨҡдәәеӣўйҳҹпјӣ
    зјәзӮ№пјҡз»“жһ„жҜ”иҫғеӨҚжқӮпјҢй—®йўҳиҝҪиёӘйңҖиҰҒйҖҗеұӮжҺ’жҹҘпјӣ
    1. еҗ„йғЁеҲҶд№Ӣй—ҙзҡ„йҖҡдҝЎпјҢйғҪжҳҜеҸҢеҗ‘зҡ„гҖӮ
    2. View дёҺ Model дёҚеҸ‘з”ҹиҒ”зі»пјҢйғҪйҖҡиҝҮ Presenter дј йҖ’гҖӮ
    3. View йқһеёёи–„пјҢдёҚйғЁзҪІд»»дҪ•дёҡеҠЎйҖ»иҫ‘пјҢз§°дёәвҖқиў«еҠЁи§ҶеӣҫвҖқпјҲPassive ViewпјүпјҢеҚіжІЎжңүд»»дҪ•дё»еҠЁжҖ§пјҢиҖҢ Presenter йқһеёёеҺҡпјҢжүҖжңүйҖ»иҫ‘йғҪйғЁзҪІеңЁйӮЈйҮҢгҖӮ

    MVVM

        Module:ж•°жҚ®еӨ„зҗҶеұӮ
    View:и§ҶеӣҫеұӮ
    ViewModule:д»ЈжӣҝPresenter,е®һзҺ°ж•°жҚ®зҡ„е®һж—¶жӣҙж–°(LiveData/DataBinding)
    1. MVVM жЁЎејҸдёӯпјҢж•°жҚ®жҳҜзӢ¬з«ӢдәҺ UI зҡ„гҖӮ
      ж•°жҚ®е’ҢдёҡеҠЎйҖ»иҫ‘еӨ„дәҺдёҖдёӘзӢ¬з«Ӣзҡ„ ViewModel дёӯпјҢViewModel еҸӘйңҖиҰҒе…іжіЁж•°жҚ®е’ҢдёҡеҠЎйҖ»иҫ‘пјҢдёҚйңҖиҰҒе’Ң UI жҲ–иҖ…жҺ§д»¶жү“дәӨйҒ“гҖӮUI жғіжҖҺд№ҲеӨ„зҗҶж•°жҚ®йғҪз”ұ UI иҮӘе·ұеҶіе®ҡпјҢViewModel дёҚж¶үеҸҠд»»дҪ•е’Ң UI зӣёе…ізҡ„дәӢпјҢд№ҹдёҚжҢҒжңү UI жҺ§д»¶зҡ„еј•з”ЁгҖӮеҚідҫҝжҳҜжҺ§д»¶ж”№еҸҳдәҶпјҲжҜ”еҰӮпјҡTextView жҚўжҲҗ EditTextпјүпјҢViewModel д№ҹеҮ д№ҺдёҚйңҖиҰҒжӣҙж”№д»»дҪ•д»Јз ҒгҖӮе®ғйқһеёёе®ҢзҫҺзҡ„и§ЈиҖҰдәҶ View еұӮе’Ң ViewModelпјҢи§ЈеҶідәҶдёҠйқўжҲ‘д»¬жүҖиҜҙзҡ„ MVP зҡ„з—ӣзӮ№гҖӮ
    2. жӣҙж–° UI
      еңЁ MVVM дёӯпјҢж•°жҚ®еҸ‘з”ҹеҸҳеҢ–еҗҺпјҢжҲ‘д»¬еңЁе·ҘдҪңзәҝзЁӢзӣҙжҺҘдҝ®ж”№пјҲеңЁж•°жҚ®жҳҜзәҝзЁӢе®үе…Ёзҡ„жғ…еҶөдёӢпјүViewModel зҡ„ж•°жҚ®еҚіеҸҜпјҢдёҚз”ЁеҶҚиҖғиҷ‘иҰҒеҲҮеҲ°дё»зәҝзЁӢжӣҙж–° UI дәҶпјҢиҝҷдәӣдәӢжғ…зӣёе…іжЎҶжһ¶йғҪеё®жҲ‘д»¬еҒҡдәҶгҖӮ
    3. еӣўйҳҹеҚҸдҪң
      MVVM зҡ„еҲҶе·ҘжҳҜйқһеёёжҳҺжҳҫзҡ„пјҢз”ұдәҺ View е’Ң ViewModel д№Ӣй—ҙжҳҜжқҫж•ЈиҖҰеҗҲзҡ„пјҡдёҖдёӘжҳҜеӨ„зҗҶдёҡеҠЎе’Ңж•°жҚ®гҖҒдёҖдёӘжҳҜдё“й—Ёзҡ„ UI еӨ„зҗҶгҖӮжүҖд»ҘпјҢе®Ңе…Ёз”ұдёӨдёӘдәәеҲҶе·ҘжқҘеҒҡпјҢдёҖдёӘеҒҡ UIпјҲXML е’Ң ActivityпјүдёҖдёӘеҶҷ ViewModelпјҢж•ҲзҺҮжӣҙй«ҳгҖӮ
    4. еҸҜеӨҚз”ЁжҖ§
      дёҖдёӘ ViewModel еҸҜд»ҘеӨҚз”ЁеҲ°еӨҡдёӘ View дёӯгҖӮеҗҢж ·зҡ„дёҖд»Ҫж•°жҚ®пјҢеҸҜд»ҘжҸҗдҫӣз»ҷдёҚеҗҢзҡ„ UI еҺ»еҒҡеұ•зӨәгҖӮеҜ№дәҺзүҲжң¬иҝӯд»Јдёӯйў‘з№Ғзҡ„ UI ж”№еҠЁпјҢжӣҙж–°жҲ–ж–°еўһдёҖеҘ— View еҚіеҸҜгҖӮеҰӮжһңжғіеңЁ UI дёҠеҒҡ A/B TestingпјҢйӮЈ MVVM жҳҜдҪ дёҚдәҢйҖүжӢ©гҖӮ
    5. еҚ•е…ғжөӢиҜ•
      жңүдәӣеҗҢеӯҰдёҖзңӢеҲ°еҚ•е…ғжөӢиҜ•пјҢеҸҜиғҪи„‘иўӢйғҪеӨ§гҖӮжҳҜе•ҠпјҢеҶҷжҲҗдёҖеӣўжөҶзіҠзҡ„д»Јз ҒжҖҺд№ҲеҸҜиғҪеҒҡеҚ•е…ғжөӢиҜ•пјҹеҰӮжһңдҪ д»¬д»Ҙд»Јз ҒеӨӘзғӮж— жі•еҶҷеҚ•е…ғжөӢиҜ•иҖҢйҖғйҒҝпјҢйӮЈеҸҜзңҹжҳҜдёҚеҘҪзҡ„ж¶ҲжҒҜдәҶгҖӮиҝҷж—¶еҖҷпјҢдҪ йңҖиҰҒ MVVM жқҘжӢҜж•‘гҖӮ
      жҲ‘д»¬еүҚйқўиҜҙиҝҮдәҶпјҢViewModel еұӮеҒҡзҡ„дәӢжҳҜж•°жҚ®еӨ„зҗҶе’ҢдёҡеҠЎйҖ»иҫ‘пјҢView еұӮдёӯе…іжіЁзҡ„жҳҜ UIпјҢдёӨиҖ…е®Ңе…ЁжІЎжңүдҫқиө–гҖӮдёҚз®ЎжҳҜ UI зҡ„еҚ•е…ғжөӢиҜ•иҝҳжҳҜдёҡеҠЎйҖ»иҫ‘зҡ„еҚ•е…ғжөӢиҜ•пјҢйғҪжҳҜдҪҺиҖҰеҗҲзҡ„гҖӮеңЁ MVVM дёӯж•°жҚ®жҳҜзӣҙжҺҘз»‘е®ҡеҲ° UI жҺ§д»¶дёҠзҡ„пјҲйғЁеҲҶж•°жҚ®жҳҜеҸҜд»ҘзӣҙжҺҘеҸҚжҳ еҮә UI дёҠзҡ„еҶ…е®№пјүпјҢйӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮдҝ®ж”№з»‘е®ҡзҡ„ж•°жҚ®жәҗжқҘй—ҙжҺҘеҒҡдёҖдәӣ Android UI дёҠзҡ„жөӢиҜ•гҖӮ
      йҖҡиҝҮдёҠйқўзҡ„з®Җиҝ°д»ҘеҸҠжЁЎејҸзҡ„еҜ№жҜ”пјҢжҲ‘д»¬еҸҜд»ҘеҸ‘зҺ° MVVM зҡ„дјҳеҠҝиҝҳжҳҜйқһеёёжҳҺжҳҫзҡ„гҖӮиҷҪз„¶зӣ®еүҚ Android ејҖеҸ‘дёӯеҸҜиғҪзңҹжӯЈеңЁдҪҝз”Ё MVVM зҡ„еҫҲе°‘пјҢдҪҶжҳҜеҖјеҫ—жҲ‘д»¬еҺ»еҒҡдёҖдәӣжҺўи®Ёе’Ңи°ғз ”гҖӮ
